# 一、什么是 vdom

  • 用 JS 模拟 DOM 结构
  • DOM 变化的对比，放在 JS 层来做
  • 提高重绘性能

# 二、设计一个需求场景

用jQuery实现

遇到的问题

  • DOM 操作是“昂贵”的，js 运行效率高
  • 尽量减少 DOM 操作，而不是“推倒重来”
  • 项目越复杂，影响就越严重
  • vdom 即可解决这个问题

# 4.1 vdom 为何使用 diff 算法

  • DOM 操作是“昂贵”的，因此尽量减少 DOM 操作
  • 找出本次 DOM 必须更新的节点来更新，其他的不更新
  • 这个“找出”的过程，就需要 diff 算法
